Group Tables

This page provides example code for working with Group Tables in the Integration Theme.

For Group Tables to be accessed via the API, they must be listed in the <permissions> block of DriveWorksConfigUser.xml and available to the current user's team.

For full configuration guidance, see Integration Theme Settings in DriveWorks Pro Help.

The shared config.js file for these examples is incomplete. Enter your own settings for the examples to function.
